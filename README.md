# Avaliação Final - Flutter


O aplicativo possui:

* Tela de Login
* Tela de Cadastro
* Tela Inicial (Home)
* Validação de Login
* Arquitetura MVVM
* Gerenciamento de estado com Provider

## Tecnologias

* Flutter
* Dart
* Provider

## Como executar

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/avaliacao_final.git
```

2. Entre na pasta do projeto:

```bash
cd avaliacao_final
```

3. Instale as dependências:

```bash
flutter pub get
```

4. Execute o projeto:

```bash
flutter run
```
